 Trabalho 3 - SSC0600 Introdução à Ciência da Computação

 Todos os conteúdos deste repositório referem-se ao desenvolvimento do Trabalho 3 proposto aos alunos de Engenharia de Computação pelo professor Adenilso da Silva Simão, na Universidade de São Paulo - Campus São Carlos.

 O Trabalho tinha como principio avaliar os conhecimentos dos alunos em programação, utilizando como base a Linguagem C. 

 Foi desenvolvido um programa capaz de operar matrizes esparsas, sendo possível realizar operações como: Criação e Remoção de Matrizes, Inclusão de Elementos, Consulta de Elementos, Soma de linhas e colunas.
